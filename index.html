AIoT Gateway API 1.0.0

Apache 2.0

The AIoT Gateway API allows you to remotely manage the end device (sensors, actuators, toys and robots).

Check out its awesome features:

  • Deploy and delete a specific project on the toy
  • Communicate between the sensor / device can be either via Wireless (MQTT) or Bluetooth 😎
  • Receive real-time information about sensor, toy, tablet and robot conditions 📈

Servers

the IP address of the aiot gateway:{port} mqtt

aiot gateway broker

port: Secure connection (TLS) is available through port 8883.
ENUM: 1883, 8883 DEFAULT: 1883
Security:
  • User / Password:

    MQTT authentication with username/password
